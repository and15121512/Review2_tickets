МИНИМАЛЬНАЯ РАБОЧАЯ ВЕРСИЯ веб-приложения "Сервис для покупки авиабилетов"
1. Запуск производился через командную строку Anacond`ы (Anaconda prompt);
Возможно, потребуется установить:
	-- flask
	-- sqlalchemy
	-- flask-sqlalchemy

Для запуска следует:
	-- перейти в корень проекта (...\Review2_)
	-- установить 'FLASK_APP=main.py' (в Windows 'set FLASK_APP=main.py')
	-- запустить приложение, прописав 'flask run'

2. После запуска:
	-- Вводятся логин и пароль пользователя (если нет, можно ввести любые доступные, в этом случае сначала будет
	предоставлена форма для регистрации)
	-- Вводятся данные, !!! поле Sum - сумма, которая будет у пользователя в распоряжении
	-- Затем переход на страницу с выбором Откуда, Куда и Дата отправления в формате DD.MM.YYYY
	-- На след. странице - результаты поиска, при нажатии на одну из панелей-результатов будет предложено купить выбранный рейс
	-- Далее при нажатии Yes билет сохраняется при достаточном количестве средств, иначе покупка совершена не будет
	-- При нажатии No происходит переход на страницу с выбором рейса

3. Для тестирования используется след. содержимое базы данных:
		Откуда   Куда    Дата отправления    Дата посадки         Стоимость
